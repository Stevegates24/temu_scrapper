# 🚀 Temu Scraper

Extract product details from Temu in one click and paste directly into Excel.

---

## ✨ Features

* 💰 Extract Price
* 🏷 Extract Brand
* 🏪 Extract Seller
* 🌍 Extract Origin
* 🚚 Extract Delivery Date

---

## 📥 Installation (1 minute)

Open [Temu Scrapper Extension Page](https://stevegates24.github.io/temu_scrapper/)

[Click here to watch](https://app.guidde.com/share/playbooks/681cM1jpMRtc6kFGDUtf3K](https://app.guidde.com/share/playbooks/681cM1jpMRtc6kFGDUtf3K?origin=j9nzieZQaXhFkdeNcSjPAV4pwPz2&mode=videoOnly))

or

1. [Download ZIP](https://github.com/Stevegates24/temu_scrapper/archive/refs/heads/main.zip)
2. Extract it
3. Open `chrome://extensions/`
4. Enable **Developer Mode**
5. Click **Load Unpacked**
6. Select the `temu_scrapper-main` folder

---

## ⚡ Usage

1. Open any Temu product page
2. Click the extension
3. Click **Copy Data (CSV)**
4. Paste into Excel

---

## 🔒 Privacy

This extension does not collect or store any data.
Everything runs locally in your browser.

---

## 🧠 Note

Wait 2–3 seconds after page loads before clicking the button.
