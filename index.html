Features

Everything you need,
nothing you don't.

Built for people who list Temu products on Amazon. No bloat, no accounts, no subscriptions — just open a product page and click.

⚡
One-click scrape
Hit Scrape Page and all fields populate instantly. The extension auto-clicks "See all details" and the shipping modal behind the scenes.
📋
Excel-ready output
Tab-separated clipboard data — Ctrl+V fills your spreadsheet columns in exactly the right order, with blanks preserved for columns you fill manually.
🏠
Local vs Non-Local
Auto-detects "Local warehouse" from page spans and tags the Warehouse column accordingly. Defaults to Non Local for standard products.
📅
Shipping dates parsed
Converts "Apr. 11–17" into MM/dd/yyyy start and end dates automatically, so your spreadsheet stays consistent.
🌙
Dark mode built-in
Toggle between light and dark with one click. Your preference is saved so it persists across every session.
🛡️
Zero data collected
Everything runs locally in your browser. No servers, no tracking, no analytics. Your product research stays yours.
How it works

Three steps.
Done.

1
Install the extension
Download the zip, unzip it, go to your browser extensions page and load unpacked. Takes under 60 seconds.
chrome://extensions → Load unpacked
2
Open any Temu product page
Navigate to a product on temu.com and wait for the page to fully load. Then click the extension icon in your toolbar.
3
Scrape → Copy → Paste
Hit Scrape Page, review the extracted data, then Copy for Excel. Click the first cell in your spreadsheet and Ctrl+V.
Ctrl+V → fills all 12 columns

Install

Get it on your browser.

Available for Chrome and Firefox. Both use the same extension package.

🌐
Google Chrome
Load unpacked via chrome://extensions or install from the Chrome Web Store once published.
→
🦊
Firefox
Submit to Firefox Add-ons (addons.mozilla.org). Same zip, fully Manifest V3 compatible.
→
Manual install (developer mode)
1. Download the .zip file from GitHub
2. Unzip into a folder
3. Go to chrome://extensions
4. Enable Developer Mode (top right)
5. Click Load unpacked → select the folder
6. Open any Temu product page and click the extension icon 🛍️
